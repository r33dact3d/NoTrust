The Big Picture

NoTrust locks payments in escrow on BSV using MNEE (1 MNEE = $1 USD). Providers deliver a service or product, buyers approve (or dispute), and funds release automatically—or we arbitrate if needed. It’s trustless: no one holds your money but the blockchain.

When Everyone’s Happy

1. Provider posts a job (e.g., “Logo, PNG, 500x500px, $50”), locks 5K MNEE arbitration fee.
2. Buyer funds the amount + 25K MNEE (20K NoTrust fee + 5K arbitration).
3. Provider delivers by deadline, buyer doesn’t dispute within 24 hours.
4. Funds release: provider gets job amount (e.g., 50 MNEE), NoTrust gets 20K MNEE, both get 5K MNEE back.

If the Buyer’s Unhappy

Buyer disputes within 24 hours, we check the specs (e.g., “PNG, 500x500px”):
- Buyer Wins: Full refund (e.g., 50 MNEE) + 5K MNEE, provider loses 5K, NoTrust gets 15K MNEE.
- Provider Wins: Job amount (e.g., 50 MNEE) + 5K MNEE, buyer loses 5K, NoTrust gets 15K MNEE.

If the Deadline’s Missed

Provider misses the deadline, buyer disputes. If undelivered, buyer wins automatically—same refund split as above.

Best Practices for Providers

- Use clear, objective specs (e.g., “JPG, 300 DPI,” not “pretty”).
- Deliver on time—missed deadlines favor the buyer.
- Upload proof (e.g., file hash) to BSV to win disputes.

Tips for Buyers

- Review specs before funding—disputes hinge on them, not opinions (e.g., “ugly” doesn’t count).
- Dispute within 24 hours of delivery, or funds release.
